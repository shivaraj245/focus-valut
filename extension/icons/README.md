# Extension Icons

Place your extension icons here:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

You can create simple icons using online tools like:
- https://favicon.io/
- https://www.canva.com/

Or use placeholder icons for now. The extension will work without custom icons (Chrome will show a default icon).

## Quick Placeholder Icons

For testing, you can use any PNG images renamed to the sizes above, or create simple colored squares using any image editor.
